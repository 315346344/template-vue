# vue初始项目模板
- TypeScript
- Router
- Vuex
- Sass
- Linter + Standard config
- axios
- element-ui


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
